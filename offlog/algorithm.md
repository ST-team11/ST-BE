# ⚡ 공과금 절약 서비스 — 알고리즘 설계 문서

---

## 서비스 개요

서울 중심의 **전기·가스·수도** 공과금 절약 서비스입니다.  
사용자가 입력한 실제 요금을 기반으로 서울시 평균 또는 전월 내역과 비교하여 절약 팁을 제공합니다.

> 💡 비회원(일회성)은 서울시 평균과 비교, 회원(이전 기록 있음)은 전월 대비 비교를 우선 적용합니다.

---

## 1. 알고리즘 전체 구조

알고리즘은 크게 세 단계로 구성됩니다.

**① 기준값 산출 → ② 비교점수 계산 → ③ 결과 유형 분류 + 절약 팁 생성**

### 비교점수 공식

```js
비교점수 = (역산된 실제 사용량) ÷ (조정된 기준값) × 100
```

| 점수 구간 | 결과 유형 | 의미 |
|---|---|---|
| 85 이하 | 절약형 🟢 | 서울 평균보다 효율적 |
| 85 ~ 115 | 평균형 🔵 | 서울 평균 수준 |
| 115 ~ 150 | 주의 🟠 | 평균 초과, 개선 여지 있음 |
| 150 초과 | 과소비형 🔴 | 유의미한 초과, 즉시 조치 필요 |

---

## 2. 입력 전처리

### 2-1. 가스비 분기

아파트 중앙난방의 경우 가스비가 관리비에 포함될 수 있습니다.

- **별도 청구**: 입력값 그대로 사용
- **관리비 포함**: 고지서 내 가스 항목 금액 별도 입력 요청

### 2-2. 수도 격월 분기

서울 수도는 격월 검침(2개월 치 청구)이 일반적입니다.

```js
월 환산 수도요금 = 고지서 금액 ÷ 검침 개월 수
```

### 2-3. 금액 → 사용량 역산

| 항목 | 역산 방식 |
|---|---|
| 전기 | 누진 구간 역산 (계절·월 기반) |
| 가스 | (청구금액 - 기본요금) ÷ 단가(원/MJ) ÷ 평균열량 ≈ 사용량(MJ) |
| 수도 | (청구금액 - 기본요금) ÷ (상수도 580 + 하수도 480 + 물이용 85.83) ≈ 사용량(톤) |

## 3. 기준값 산출 질문 (Q1·Q3 필수)

### Q1 — 가구원 수

> 가구원 수는 몇 명인가요? · 1명 · 2명 · 3~4명 · 5명 이상

**영향 항목**: 전기·가스·수도 모두

| 답변 | 전기 보정계수 | 수도 서울 평균 (월) |
|---|---|---|
| 1명 | ×1.5 | 8톤 / 약 9,480원 |
| 2명 | ×1.0 (기준) | 13톤 / 약 14,730원 |
| 3~4명 | ×0.85 | 16~18톤 / 약 17,880~19,980원 |
| 5명 이상 | ×0.75 | 21톤 이상 / 약 23,130원+ |

> 서울시 에코마일리지 데이터 기반: 1인 가구의 1인당 에너지 소비는 5인 가구 대비 **3.8배** 수준

```js
전기 기준값(kWh) = 서울평균(275kWh/월) × 면적보정(Q3) × 가구원수보정(Q1)
```

### Q3 — 주거 면적

> 주거 면적(평수)은 어느 정도인가요? · 10평 이하 · 10~20평 · 20~30평 · 30평 이상

**영향 항목**: 전기·가스

| 답변 | 전기 면적보정 | 가스 동절기 기대범위 |
|---|---|---|
| 10평 이하 | ×0.70 | 5~8만원 |
| 10~20평 | ×1.00 (기준) | 8~12만원 |
| 20~30평 | ×1.25 | 12~18만원 |
| 30평 이상 | ×1.55 | 18만원+ |

> 수도는 면적 영향이 미미(40평↑에서도 약 8% 차이)하므로 보정 생략

## 4. 보조 보정 질문 (Q2·Q4·Q12)

### Q2 — 거주 형태

> 거주 형태는 무엇인가요? · 원룸 · 아파트 · 빌라 · 오피스텔 · 기타

**영향 항목**: 가스 / 결과 해석 맥락

| 답변 | 처리 방식 |
|---|---|
| 아파트 | 중앙난방 여부 확인 → 가스비 관리비 포함 분기 트리거 |
| 원룸·빌라·오피스텔 | 개별난방 기본값으로 취사+난방 겸용 요금 적용 |
| 기타 | 가스 결과 신뢰도 낮음 표시 |

### Q4 — 거주 지역

> 거주 지역은 어디인가요? · 서울 · 수도권 · 남부 지역 · 제주 · 기타

**영향 항목**: 전기·가스·수도 모두

| 답변 | 처리 방식 |
|---|---|
| 서울 | 서울 평균 기준값 정상 적용 |
| 수도권 | 서울 기준 근사 적용 + 단가 차이 안내 |
| 남부·제주·기타 | 전기만 정확 비교, 가스·수도는 참고값으로 표시 |

### Q12 — 건물 연식 (선택)

> 건물 연식은 어느 정도인가요? · 5년 이하 · 5~15년 · 15년 이상

**영향 항목**: 가스(난방 효율)

| 답변 | 가스 기준값 보정 | 추가 처리 |
|---|---|---|
| 5년 이하 | ×0.85 | - |
| 5~15년 | ×1.0 (기준) | - |
| 15년 이상 | ×1.2 | 결과에 "단열 점검 권장" 팁 추가 |

## 5. 절약 잠재값 산출 질문 (Q5~Q11·Q13)

### Q5·Q6 — 에어컨 사용 (여름 한정)

> Q5. 여름에 에어컨을 하루 평균 몇 시간 사용하시나요?  
> Q6. 에어컨 온도는 보통 몇 도로 설정하시나요?

**영향 항목**: 전기

| Q5 사용 시간 | 전기 가산 추정 |
|---|---|
| 거의 사용 안함 | +0 kWh/월 |
| 1~2시간 | +54 kWh/월 |
| 3~5시간 | +144 kWh/월 |
| 6시간 이상 | +216 kWh/월 |

> 에어컨 평균 소비전력 1.2kW 가정 (30일 × 사용시간 × 1.2kW)

| Q6 설정 온도 | 소비 보정 |
|---|---|
| 26도 이상 | ×0.8 (절약) |
| 24~26도 | ×1.0 (기준) |
| 23도 이하 | ×1.3 (과소비) |

```js
에어컨 소비(kWh/월) = 시간별 기준값 × 온도보정
절약 여지(원) = 에어컨 소비 × (1 - 0.8) × 해당 누진 단가
→ "온도 1도 올리면 월 약 N원 절약" 메시지 생성
```

### Q7 — 겨울 난방 패턴 (겨울 한정)

> 겨울에 난방은 보통 어떻게 사용하시나요? · 거의 사용하지 않음 · 외출 시 끔 · 하루 대부분 켬

**영향 항목**: 가스 (전기난방 보유 시 전기도 해당)

| 답변 | 가스 소비 추정 | 처리 방식 |
|---|---|---|
| 거의 사용 안함 | 기대값의 40% | 가스 높으면 누수/기기 이상 의심 팁 |
| 외출 시 끔 | 기대값의 80% | 절약 유형 분류 + 긍정 피드백 |
| 하루 대부분 켬 | 기대값의 120% | 외출 시 절감 가능 금액 제시 |

```js
가스 절약 여지 = 실제가스비 × 난방패턴_과소비율 × 0.6
(행동 요인 기여 추정 60%)
```

### Q8 — 대기전력 관리

> 사용하지 않는 가전제품의 전원은 어떻게 관리하시나요?

**영향 항목**: 전기

| 답변 | 대기전력 추정 | 월 절약 가능액 |
|---|---|---|
| 멀티탭 꺼둔다 | ~5W | 절약 실천 중 → 다른 팁 제시 |
| 그냥 꽂아둔다 | ~50~80W | 약 4,300~6,800원 절약 가능 |
| 잘 모르겠다 | 중간값 적용 | 약 3,300원 잠재 |

```js
대기전력 절약 여지(kWh/월) = 대기전력(W) × 24h × 30일 ÷ 1000
절약 금액 = 절약 kWh × 해당 누진 구간 단가
```

> 한국에너지공단 기준: 가정 평균 대기전력 약 11% 수준

### Q9 — 조명 사용 패턴

> 집에서 조명을 얼마나 켜두나요?

**영향 항목**: 전기

| 답변 | 추정 소비 | 절약 여지 |
|---|---|---|
| 필요한 곳만 | ~10 kWh/월 | 절약 실천 중 |
| 대부분 켜둔다 | ~25 kWh/월 | ~15 kWh → 약 1,800원 |
| 하루 종일 | ~40 kWh/월 | ~30 kWh → 약 3,600원 |

> LED 기준 가구당 조명 총 100W 가정. 개별 절약 금액보다 전체 절약 점수에 합산 처리 권장

### Q10 — 요리 빈도

> 집에서 요리는 얼마나 자주 하시나요?

**영향 항목**: 가스

| 답변 | 취사 비중 추정 | 처리 방식 |
|---|---|---|
| 거의 안함 | 약 10% | 가스비 주 원인은 난방 → 난방 중심 분석 |
| 하루 1번 | 약 20% (기준) | - |
| 하루 2번 이상 | 약 35% | "요리 빈도 고려 시 정상" 맥락 메시지 |

```js
취사 가스비 = 전체 가스비 × 요리빈도 비중
난방 가스비 = 전체 가스비 - 취사 가스비
→ 취사 / 난방 분리 비교
```

### Q11 — 평일 낮 재택 시간

> 평일 낮에 집에 머무는 시간은 어느 정도인가요?

**영향 항목**: 전기·가스

| 답변 | 전기 기준값 보정 | 메시지 처리 |
|---|---|---|
| 거의 없음 | ×0.85 | 높으면 대기전력 의심 팁 |
| 3~6시간 | ×1.0 (기준) | - |
| 하루 대부분 | ×1.25 | "재택 감안 시 평균 수준" 안내 |

> Q8(대기전력) + Q11(재택) 조합: "집에 거의 없는데 전기요금 높다" → 대기전력·누수 의심 팁 강조

### Q13 — 보유 가전 (선택)

> 집에 있는 주요 가전을 모두 선택해주세요.

**영향 항목**: 전기

| 가전 | 월 소비 추정 | 처리 방식 |
|---|---|---|
| 건조기 | +30~50 kWh | 전기 기준값에 가산 후 비교 |
| 식기세척기 | +10~15 kWh | 동일 |
| 공기청정기 | +5~10 kWh | 동일 |
| 전기난방 | +60~120 kWh (겨울) | 기준값 상향 + 가스는 낮아도 정상 처리 |

```js
조정 기준값(kWh) = 기본 기준값 + Σ 보유 가전 소비 추정값
```

## 6. 최종 계산 공식 전체

### 전기

```js
기준값(kWh) = 275 × 면적보정(Q3) × 가구원수보정(Q1) × 재택보정(Q11) + Σ 가전가산(Q13)
비교점수_전기 = 실제사용량(역산) ÷ 기준값 × 100
```

### 가스

```js
기준값(가스) = 면적별_계절_기대값 × 가구원수보정(Q1) × 건물연식보정(Q12) × 난방패턴보정(Q7)
취사보정 = 기준값 × 요리빈도_비중(Q10)
비교점수_가스 = 실제가스비 ÷ (기준값 × 단가환산) × 100
```

### 수도

```js
기준값(수도) = 가구원수별_서울평균(Q1)
비교점수_수도 = 실제사용량(역산) ÷ 기준값 × 100
```

### 절약 잠재점수

```js
절약점수 = Σ(대기전력Q8 + 조명Q9 + 에어컨Q5·Q6 + 난방Q7) - 각 0~25점
70점 이상 → 절약 팁 많음
40~70점   → 보통
40점 미만  → 이미 잘 절약 중
```

## 10. 결과 유형 상세 분류 (10종)

### 설계 원칙

결과 유형은 두 축의 교차로 결정됩니다.

- **축 1 - 항목 패턴**: 전기·가스·수도 중 어느 항목이 높은가
- **축 2 - 습관 패턴**: 생활습관 질문(Q5~Q13)에서 도출된 주요 원인 행동

### 유형 판별 흐름

```js
[전기 비교점수] + [가스 비교점수] + [수도 비교점수]
          +
[절약잠재점수 구성 - 어느 항목이 높은가]
          ↓
10개 유형 중 1개 매칭
```

## 11. 구현 시 주의 사항

1. **가스 역산의 한계** — 온압보정계수·월평균열량이 지역·월별로 변동. 고지서 금액을 그대로 비교하는 방식이 더 현실적일 수 있음
2. **수도 격월 청구** — UI에서 "이번 고지서가 몇 개월치?" 확인 필수
3. **가스 면적별 평균 데이터 부재** — 공개 통계 없음. 결과 표시 시 부가 설명 병행
